1. W tej pierwszej misji wystarczy zadeklarować dwie zmienne a, b i ustawić ich wartości odpowiednio na liczby całkowite 2, 5.

2. Po zakończeniu proszę spróbować ustawić wartości zmiennych w jednej linii (pierwsza zmienna otrzymuje pierwszą wartość itd.).

Podczas gdy Python pozostaje językiem dynamicznie typowanym, co oznacza, że nie trzeba określać typów dla zmiennych, włączenie adnotacji typu, szczególnie w większych projektach lub podczas pracy w zespole, może znacznie poprawić jakość kodu i łatwość konserwacji. Warto zauważyć, że adnotacje typów są opcjonalne, a Python nie wymusza ścisłego typowania; służą one przede wszystkim jako pomoc programistyczna i narzędzie do dokumentacji.

3. Dla liczb całkowitych proszę użyć typu int. Proszę przepisać pierwszy wariant z adnotacją typu.

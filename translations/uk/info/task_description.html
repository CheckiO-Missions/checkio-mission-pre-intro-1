1. У цій найпершій місії вам потрібно лише оголосити дві змінні a, b та встановити їх значення цілими числами 2, 5 відповідно.

2. Після цього спробуйте задати значення змінних в одному рядку (перша змінна отримує перше значення і т.д.).

Хоча Python залишається динамічно типізованою мовою, тобто вам не потрібно вказувати типи для змінних, включення анотацій типів, особливо у великих проектах або при роботі в команді, може значно підвищити якість коду та його ремонтопридатність. Варто зазначити, що анотації типів є необов'язковими, і Python не вимагає суворої типізації; вони слугують, насамперед, як допоміжний інструмент для розробки та документування.

3. Для цілих чисел використовуйте тип int. Перепишіть перший варіант з анотацією типів.
